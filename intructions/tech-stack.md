
## Technical Architecture

### Frontend
- **Framework**: NextJS 15
- **Language**: TypeScript
- **Styling**: TailwindCSS, ShadCN UI, Radix UI
- **State Management**: Next.js App Router, SWR for data fetching

### Backend
- **Runtime**: Cloudflare Workers
- **Framework**: Hono
- **Database**: PostgreSQL (Neon)
- **ORM**: Drizzle ORM
